## Description

Let's build something from nothing using this template <br />

## what this template do?

- Import Asset (svg/video) without any additional configuration
- Path alias work like charm

## Installation

```bash

npm install

```

## Running the app

```bash
# Build app
npm run build

# Development
npm run start

```
